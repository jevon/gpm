How It Works

GPM's unified server delivers deep, AI-powered research for any package your AI assistant needs to understand.

1

Start the Unified Server

Run gpm serve to start the unified MCP server that can dynamically research any package.

2

Request Package Info

AI assistants query the server for information about any package through RESTful endpoints.

3

Deep Research

The server performs extensive AI-powered research, analyzing registry data, GitHub repositories, and extracting key examples.

4

Intelligent Caching

Research results are cached for performance while still maintaining the ability to refresh when needed.

5

Optimized Response

Context is tailored to the detected AI assistant, ensuring the most effective format for each tool.
